# 你好，世界！

欢迎来到我的个人博客！👋

这是我用 **Astro** 搭建的第一个博客网站，也是我的第一篇正式文章。

## 为什么要写博客？

我一直想有一个属于自己的空间，用来：

- 📚 **记录学习历程** — 把学到的知识点整理成文，加深理解
- 💡 **分享心得体会** — 学习过程中遇到的问题和解决方法
- 🛠️ **展示项目作品** — 展示自己做的小东西和实验
- 🔗 **资源推荐** — 分享好用的工具、教程和书籍

## 搭建这个博客用了什么？

| 技术 | 说明 |
|------|------|
| **Astro** | 2025 年最前沿的静态网站生成器，性能极佳 |
| **Markdown** | 写文章用的标记语言，简洁高效 |
| **GitHub** | 代码托管，版本管理 |
| **Vercel** | 自动部署，免费托管 |
